Interactive TO-DO List

A dynamic, browser-based TO-DO List built with HTML, CSS & JavaScript. This project is all about making productivity fun and interactive!  

Features

- Add & Delete Tasks – Quickly add new tasks or remove completed ones.  
- Mark Tasks Complete – Check a box and watch tasks get a satisfying line-through.  
- Persistent Storage – Tasks are saved in **localStorage**, so they stay even after refreshing.  
- Clean & Responsive Design – Looks great on any screen with smooth interactive UX.  

How to Use

1. Open `index.html` in your browser.  
2. Add your tasks in the input field.  
3. Check off tasks as you complete them.  
4. Delete tasks you no longer need.  

Tech Stack

- HTML  
- CSS  
- JavaScript
